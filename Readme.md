# AI-Based Rockfall Prediction & Early Warning System

## Overview

This project predicts rockfall hazards in open-pit mines using:

- Sensor Monitoring
- Terrain Analysis
- Weather Analysis

The outputs are fused to generate a final risk score and alert level.

## Features

- Sensor Risk Prediction
- Terrain Risk Analysis
- Weather Risk Analysis
- Final Risk Fusion
- Risk Gauge Dashboard
- Prediction History
- Downloadable Reports

## Models

### Sensor Model
Random Forest Classifier

### Terrain Model
Isolation Forest

### Weather Model
Isolation Forest
